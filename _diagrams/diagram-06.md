**Walk-through:** Five anti-patterns on the left (red), each with its corresponding fix on the right (green). Every fix is a configuration change — no application code changes required. The arrows show the direct mapping from problem to solution.

**Audience interaction:** Ask for a show of hands on each anti-pattern. "How many of you have hardcoded -Xmx?" Usually half the room.
